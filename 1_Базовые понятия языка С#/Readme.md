# Лабораторная работа: 1. Базовые понятия языка С#

## Автор: Кузьмин Вячеслав Сергеевич
## Группа: ПИ-251

## Цель работы
Освоение базового синтаксиса C# и математических вычислений.

## Описание
Программа выполняет два задания:
1. Вычисление a^n, используя только умножение.
2. Преобразование числа: удаление второй цифры и приписывание ее справа.

## Исходные данные
- Для первого задания: целые числа a и n
- Для второго задания: целое число x (x >= 100)

## Ожидаемые результаты
- Для первого задания: значение a^n
- Для второго задания: преобразованное число n

## Пример работы
Основание: 2
Степень: 3
2^3 = 8

Число (≥100): 121111
Результат: 111112
